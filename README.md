# Simple CloudOps Dashboard

A minimal dashboard demonstrating DevOps, AIOps, GreenOps, and Cloud concepts.

## Features
- DevOps: CI/CD simulation, deployment metrics
- AIOps: Anomaly detection, auto-remediation
- GreenOps: Energy optimization, carbon tracking
- Cloud: Multi-cloud status monitoring

## OpenShift Deployment
1. Import this repository in OpenShift Console
2. Use Docker strategy
3. Access via created route

## Local Testing
Simply open `index.html` in any browser.
